## Template Usage Guide / Şablon Kullanım Kılavuzu

### Nasıl Kullanılır?

1. **Her cycle sonunda güncelle** — Ne yapıldı, hangi kararlar alındı
2. **Next Action her zaman dolu olmalı** — Boşluk = takip edilecek iş yok
3. **Active Projects tablosu canlı tut** — Durum değişince güncelle
4. **Open Questions sorunsuz kapanır** — Cevap bulduğunu sil

###何时更新？

- Cycle başında — Next Action define et
- Cycle sonunda — What We Did + Key Decisions doldur
- Yeni proje başladığında — Active Projects tablosuna ekle
- Blokaj olduğunda — Open Questions ekle

### Color Coding (Notion'da)

- 🟢 IN PROGRESS — Aktif çalışma
- 🟡 BACKLOG — Yakında
- 🔴 PAUSED — Blokaj
- ✅ COMPLETED — Tamamlandı
