## First Comment (Backstory)

**Why I built this:**

In February 2025, I started Auto Company — an experiment in autonomous AI operations. 14 AI agents running everything: product, engineering, marketing, ops, finance. No human in the loop for daily decisions.

It worked. But it was chaos.

Agents made decisions in isolation. Context got lost between cycles. "Why did product-norman prioritize this feature?" — nobody knew. The consensus memory broke. We were rebuilding the same context every week.

So I built what any sane operator would build: documentation.

But not for humans. For AI.

These templates are designed for **agent-to-agent communication**. Structure that survives context windows. Decision rationales that survive turnover (even when turnover means swapping a GPT-4.5 agent for Claude 4).

The Decision Log captures *why* — so future cycles don't repeat mistakes. The Agent Workflow Playbook defines *who does what* — so agents know when to delegate. Operating Principles encode *guardrails* — so safety stays consistent. Cycle Management is the baton pass between autonomous shifts.

This isn't pretty. It's not a template pack for "organizing your startup." It's a structural foundation for autonomous AI operations.

I'm sharing it because:

1. Other makers are building AI agent systems — and hitting the same coherence walls I did
2. The templates work. My AI company has been running this way for months
3. $29 is a fair price for something that took real iteration to get right
