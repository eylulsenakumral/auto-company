## 5. Data Ownership & Privacy (Veri Sahipliği ve Gizlilik)

### 5.1 Data Ownership (Veri Sahipliği)

**Factory Data Rights (Fabrika Veri Hakları):**
- Party B retains full ownership of all data generated during pilot
- Data includes: video footage, alert logs, incident reports, metrics
- Export rights: Party B can request full data export in CSV/JSON/MP4 format at any time
- No data lock-in: Pilot cancellation does not affect data access
- Export requests processed within 3 business days (standard) or 24 hours (priority, for converting customers only)

**NextVision Data Rights (NextVision Veri Hakları):**
- License to use anonymized, aggregated data for product improvement
- License to use case study testimonials (Party B name optional)
- **No right** to sell or share Party B's data with third parties
- **No right** to use Party B's data for AI model training without explicit consent

### 5.2 Privacy & Security (Gizlilik ve Güvenlik)

**Data Storage (Veri Depolama):**
- Edge processing: Video processed locally, never leaves factory premises (unless cloud sync enabled)
- Cloud storage: Encrypted at rest (AES-256), encrypted in transit (TLS 1.3)
- Data retention: 30 days during pilot, exportable on request
- Data deletion: Party B can request immediate deletion via support ticket (processed within 24 hours)

**Compliance (Uyumluluk):**
- GDPR compliant (EU data residency available upon request)
- KVKK compliant (Turkish Law No. 6698 on Protection of Personal Data)
- SOC 2 Type II certified cloud infrastructure (available upon request for Enterprise plan)

**Anonymization (Anonimleştirme):**
- Facial blur enabled by default (identifiable faces not stored)
- Vehicle license plates blurred unless vehicle module active
- Worker identity tracking opt-in (badge integration required for individual tracking)
- No biometric data collection without explicit consent

### 5.3 Data Export & Portability (Veri Dışa Aktarma)

**Export Formats (Dışa Aktarma Formatları):**
- CSV: Alert logs, incident reports, metrics
- JSON: Raw data for integrations
- MP4: Video clips (annotated alerts only, not raw footage)

**Export Process (Dışa Aktarma Süreci):**
- Standard request: 3 business days (email to support)
- Priority request: 24 hours (available to converting customers only)
- Export size limit: 10GB per request (split into multiple requests for larger datasets)
