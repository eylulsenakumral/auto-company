# Product Hunt Launch Tool - GitHub Pages Deployment

## 2 Dakikalık Manual Setup

### Adım 1: GitHub Repository'ya Git
URL: https://github.com/eylulsenakumral/product-launch-tool

### Adım 2: Pages Enable
1. **Settings**'e tıkla
2. Sol menüden **Pages**'i bul
3. **Source** kısmında **GitHub Actions**'ı seç
4. **Save** butonuna tıkla

### Adım 3: Otomatik Deployment
- Workflow otomatik tetiklenecek
- 2-3 dakika sonra site live olacak
- URL: `https://eylulsenakumral.github.io/product-launch-tool/`

### Adım 4: Test

- Preview gör
- Export test et
- Auto-save kontrol et
