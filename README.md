# Auto Company Demo Package

4 Product Demos — No Installation Required

## What's Inside

1. **LeadQualifier** — B2B Lead Scoring API
   - Qualify leads instantly with scoring algorithm
   - REST API simulation

2. **MockFirst** — Local API Mock Server
   - Develop frontend without backend dependency
   - CLI-based mock server

3. **Smoke Test Landing Pages** — Week 1 Validation
   - Pre-built landing page templates
   - Validate ideas before building

4. **Changelog Generator** — GitHub Release Automation
   - Auto-generate changelogs from commits
   - Streamline release process

## How to Use

### Option 1: Demo Portal (Recommended)
1. Open `DEMO_PORTAL.html` in your browser
2. Click any product card to view its demo
3. Navigate between products easily

### Option 2: Individual Demos
1. Navigate to any product folder
2. Open `demo.html` directly

## Requirements

- Modern browser (Chrome, Firefox, Safari, Edge)
- No server or installation required
- Works entirely offline
